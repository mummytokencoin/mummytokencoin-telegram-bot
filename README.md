# MUMMYTOKENCOIN Telegram Bot

Telegram bot for MUMMYTOKENCOIN ($MTC) - LHOPE Memorial NFTs.

## Commands

- `/start` - Welcome message with all links
- `/website` - Main website
- `/mint` - NFT minting DApp
- `/lhope` - LHOPE Fund wallet
- `/mtc` - $MTC token contract
- `/donate` - Donation instructions
- `/info` - Project overview

## Links

- Bot: https://t.me/MUMMYTOKENCOIN_bot
- Website: https://mummytokencoin.com
- Mint DApp: https://mint.mummytokencoin.com

## Deployment

This bot is deployed on Render.com (free tier).
